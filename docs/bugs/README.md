# docs/bugs/

Pasta para documentar bugs resolvidos em produção.

## Formato de arquivo

Crie um arquivo `BUG-NNN.md` para cada bug relevante:

```
BUG-001.md — Descrição curta do bug
BUG-002.md — ...
```

## Template

```markdown
# BUG-NNN — Título do Bug

**Data:** AAAA-MM-DD
**Severidade:** Crítico | Alto | Médio | Baixo
**Status:** Resolvido | Investigando

## Sintoma

O que o usuário via / o que estava errado.

## Causa raiz

Por que acontecia.

## Solução

O que foi feito para resolver. Arquivos modificados.

## Como prevenir

O que fazer para isso não voltar a acontecer.
```
