## Problemas encontrados e soluções

1. **Emoji/unicode no terminal Windows** — Script Python falhava com `charmap` codec error. Solução: abrir stdout com encoding UTF-8 explícito.
2. **Path separator no Windows** — Script de verificação de paths usava `/` mas os paths do Windows usam `\`. Solução: normalizar com `filepath.replace(chr(92), '/')`.
3. **Lenis CDN duplicado** — Após refatoração, o script CDN aparecia duas vezes. Solução: regex específico para remover apenas a primeira ocorrência.
4. **Git staged deletions pré-existentes** — Arquivos na raiz do projeto (bank.png, etc.) estavam marcados como `D` no git antes desta sessão. Stagados separadamente.
