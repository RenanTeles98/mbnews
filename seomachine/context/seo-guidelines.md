# MB Finance — Diretrizes SEO

## Estrutura do Artigo
- **H1**: Contém a palavra-chave primária. Máximo 60 caracteres.
- **Introdução**: Palavra-chave nos primeiros 100 palavras. Use o gancho APP (Agree, Promise, Preview).
- **H2**: 4-7 seções. Pelo menos 2 devem conter variações da palavra-chave.
- **Comprimento**: 1.800-2.500 palavras para artigos informativos. 1.200-1.500 para comparativos.
- **Meta Title**: 50-60 caracteres com palavra-chave primária.
- **Meta Description**: 150-160 caracteres. CTA claro.

## Densidade de Palavras-chave
- Palavra-chave primária: 1-2% do texto
- Variações/LSI: distribuídas naturalmente
- Evitar keyword stuffing

## Links
- **Internos**: 3-5 por artigo apontando para páginas do site MB Finance
- **Externos**: 2-3 por artigo para fontes de autoridade (Banco Central, SEBRAE, G1 Economia)

## Frontmatter Obrigatório no Rascunho
Sempre incluir no início do artigo gerado:
```
**Title**: Título do artigo
**URL Slug**: slug-do-artigo
**Meta Title**: Meta título SEO (50-60 chars)
**Meta Description**: Descrição para Google (150-160 chars)
**Target Keyword**: palavra-chave principal
**Excerpt**: Resumo de 1-2 frases para o card do blog
**Category**: categoria (credito | conta-pj | gestao | antecipacao | mercado | noticias)
**Read Time**: X min
```

## Páginas Internas para Linkar
- /mb-finance-completo.html — página principal
- /blog — listagem do blog  
- /sobre.html — sobre a empresa
- Artigos relacionados em /blog/[slug]

## Tom nos Títulos
- Use números: "7 formas de...", "5 erros que..."
- Use urgência/relevância: "em 2025", "para PMEs"
- Perguntas diretas: "Como conseguir crédito para sua empresa?"
- Benefício claro: "sem burocracia", "em minutos", "sem análise"
